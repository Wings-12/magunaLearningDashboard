# magunaLearningDashboard

Tailwind CSS と Chart.js で作成した学習ダッシュボードのモックアップです。  
`docs/index.html` が GitHub Pages で公開されるエントリーポイントになっています。

## ローカルで確認する

ブラウザで `docs/index.html`（もしくは `test/uiDesign_reality2.html`）を直接開くだけで表示できます。開発サーバーなどは不要です。

## GitHub Pages で公開する手順

1. このリポジトリを GitHub 上に push します（例: `magunaLearningDashboard`）。
2. GitHub のリポジトリページで **Settings > Pages** に移動します。
3. **Build and deployment** で **Source: Deploy from a branch** を選択します。
4. **Branch** を `main`、**Folder** を `/docs` に設定し **Save** します。
5. 数分後に `https://<GitHubユーザー名>.github.io/magunaLearningDashboard/` が公開されます。

カスタムドメインを利用したい場合は、同じ Pages 設定画面で `Custom domain` を登録してください。
